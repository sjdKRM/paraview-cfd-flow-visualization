# Resume / LinkedIn Blurb

## Project title

**CFD-Style Flow Visualization in ParaView Using a Reproducible Python Pipeline**

## LinkedIn project description

Built a reproducible scientific visualization workflow using ParaView and Python automation. The project loads a CFD-style VTK dataset, creates scalar slice views, contour/threshold visualizations, vector-field streamlines, tube-rendered flow paths, exported screenshots, a saved `.pvsm` state file, and an MP4 camera-orbit animation. I documented the workflow for AI-training-style evaluation, including expected outputs, common model mistakes, and validation criteria.
