## Title

CFD-Style Flow Visualization in ParaView Using a Reproducible Python Pipeline

## One-line summary

I built a no-GUI ParaView project that loads CFD-style flow data, creates scalar and vector visualizations, exports screenshots, saves a `.pvsm` state file, and generates an MP4 camera-orbit animation.

## LinkedIn project description

Built a command-line scientific visualization workflow in ParaView using `pvpython`. The project loads a CFD-style VTK dataset, applies scalar slicing, contour/threshold visualization, vector-field streamlines, tube rendering, color mapping, screenshot export, `.pvsm` state saving, and MP4 animation creation. I also documented validation notes for reviewing AI-generated ParaView workflows, including common mistakes such as confusing scalar/vector fields, using streamlines without a vector field, and treating `.pvsm` files as video outputs.
